# Exercise 1
 
Write a Python script to concatenate the following dictionaries to create a new one.


Example input 1:
```python
dic1 = {1: 10, 2: 20}
dic2 = {3: 30, 4: 40}
```
  Expected Result : ```{1: 10, 2: 20, 3: 30, 4: 40}```


Example input 2:
```python
dic1 = {1: 10, 2: 20}
dic2 = {3: 30, 4: 40}
dic3 = {5: 50, 6: 60}
```
  Output : ```{1: 10, 2: 20, 3: 30, 4: 40, 5: 50, 6: 60}```
